About

Code Breaker 5.10 is derived from a game called Bulls and Cows of unknown origin. It's game principle is public domain. There are commercial versions of Bulls and Cows from other parties using different names or trademarks being published, distributed, and available as well. Code Breaker 5.10 is MIT licensed and just uses the public domain principle of the game Bulls and Cows. It might come bundled with software using free software licensing. While other commercial versions might cover technical solutions and principles that are protected by patents or stricter law in general. Be aware if you reuse any parts of this project not to infringe such protected intellectual property. The by far most successful commercial version is most likely the classic board game invented by Mordecai Meirowitz in 1970. Anyway all these variants resemble Bulls and Cows, too, with slightly adapted rules, other data representation, or modified more or less sophisticated gaming material.


Rules

The rules of Code Breaker 5.10 are as follows.

  • Iteratively a guess is stated to be compared towards a secret five digit code with ten possible values per digit (letters from a to j).
  • Objective is to match the secret code perfectly indicated by a result stating
    > 5,0.
  • On each guess a result indicates the amount of perfect matches and imperfect matches represented as numerical values.
    • Perfect matches indicate that a digit is included in the secret and is placed at correct position inside the guess.
    • Imperfect matches indicate that a digit is included in the secret but the position is still incorrect.
  • Thus the secret code is found if five perfect matches and no imperfect match are indicated on a check.
  • Digit values may occur multiple times. No digit of the guess will count multiple times in the indicated result. Indicating perfect matches over imperfect ones is prioritized. Previous check results are kept visible in a log to use for analysis.

This is the plain vanilla HMI version of Code Breaker 5.10 only.

Please mind that... entering a guess is done on the native keyboard of your device. User experience for typing the guess depends on the input method and personal settings of your device, e.g. auto correction settings.
