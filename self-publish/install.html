Why am I getting an Install failed, error: MULTIPLE_APPS_PER_ORIGIN_FORBIDDEN on installing the applications?
This simply means you already installed an application referring to or hosted on the same URL base path. The term ORIGIN does not exactly match a set of components from a URI. On Mozilla Developer Network (MDN) the term ORIGIN is therefore explained by examples given.
  • This application you installed could possibly be a different application but from the same URL base path. In other words it is not neccessarily the same application that is already installed.
  • To get more detailled explanation on this security related matter and deep dive into the subject visit
    • MDN - Apps.install
    • MDN - Same origin policy for JavaScript
    • Mind differences on file: protocol URIs - MDN - Same origin policy for file: URIs
    • W3C - Same Origin Policy
So how to proceed?
  • First of all check whether the application of your choice is not already installed. If so then enjoy the application!
  • If you are sure the application of your choice is not available on your device then
    • one choice might be that the conflicting other application has to be removed before installing the intended application or
    • you could download the newly intended application from its source repository or home page.
    • On a FirefoxOS enabled device any downloaded application fully available in source can be uploaded using the corresponding feature from inside the FirefoxOS Simulator onto the connected target device.
    • Alternatively it is possible to visit the online version of the application if available from its home page and use your browser's offline.appcache to store the whole application for offline usage. In that case you have to enable and allow the offline storage for your browser.
